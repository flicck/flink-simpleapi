# flink-simpleapi
由于笔者觉得flink的一些api太难记了，就写了这个工具包，将一些函数放在一块方便大家调用。  
使用方式：maven clean install  
然后在其他项目中加入  
 
`<dependency>`  
`<groupId>io.flicck.github</groupId>`  
`<artifactId>flink-simpleapi</artifactId>`  
`<version>1.0.0</version>`  
`</dependency>`
